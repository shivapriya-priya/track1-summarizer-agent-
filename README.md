#  AI Summarizer Agent (Track 1 - GenAI Hackathon)

##  Overview
This project is an AI agent built using Google ADK (Agent Development Kit).  
It performs text summarization by generating a short title and concise summary from user input.

##  Features
- Summarizes long text into short meaningful content
- Generates a title + summary
- Built using Google ADK
- Simple and efficient logic

##  Tech Stack
- Python
- Google ADK
- Cloud-ready architecture
